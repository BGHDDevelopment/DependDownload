BGHDDevelopment Depends & Addons

Here you can find all the depends and addons for our plugins.

FateUHC

  • Used to enable BossBars in FateUHC.
  • Used to enable Bungee Announce in FateUHC.
  • Used to enable Holograms in FateUHC.
  • Used to enable the Anonymous scenario in FateUHC.

FateMeetup

  • Required to run FateMeetup.
  • Used to enable Bungee Announce in FateMeetup.
  • kits.yml which includes more kits.

FateHub

  • Required to use Mizu Core with FateHub.
